CSS Position Explained

CSS position property defines how an element is positioned in the document. Here are the common values:

Static (Default)

The default position. Elements are placed in the normal document flow.

Static

Relative

The element is positioned relative to its normal position. Offsets can be applied using top, right, bottom, and left.

Relative

Absolute

The element is positioned relative to its nearest positioned (non-static) ancestor or the document if none exist.

Absolute

Fixed

The element is positioned relative to the viewport. It does not move when the page is scrolled.

Fixed

Sticky

The element toggles between relative and fixed, based on the user's scroll position. It sticks to a defined offset but moves with the content until a boundary is reached.

Sticky
